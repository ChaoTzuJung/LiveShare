# LiveShare

**2017國立臺北教育大學 Web&App創作競賽 - Web企劃&成品實作組 第二名**

Warn: Project crash and pending 😰😞 ... 

## Summary
這個網頁是一個教學平台，最一開始的發想是想要把繫上講師講課的內容錄製成影片放上平台供大家課後複習，最後我們的定位方向是把他當作是一個可以開課的平台，希望不只是老師，繫上的學長姐也能夠在這個平台開課，展謝自己的專長與訓練自己的表達能力。

p.s. 靈感來自 @Hahow
